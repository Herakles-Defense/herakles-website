# Herakles Defense - Official Website

Official website for Herakles - Tactical Counter-UAS Defense System.

## Technology Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
- **Domain:** herakles-defense.com

## Corporate Design

- **Primary Color:** #18181B (Fast-Schwarz)
- **Secondary Color:** #27272A (Dunkelgrau)
- **Accent Color:** #F97316 (Signal-Orange)
- **Neutral Color:** #FAFAFA (Fast-Weiss)
- **Style:** Tactical Minimalism

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Structure

- `/app` - Next.js App Router pages
- `/components` - Reusable React components
- `/public/images` - Static images and assets

## Deployment

Automatically deployed to Vercel on every push to main branch.

Production URL: https://herakles-defense.com
